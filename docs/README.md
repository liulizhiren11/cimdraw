# 文档入口

`docs/` 目录当前分为两类内容：

1. `design/`
   面向 `lz-power` 项目落地的设计文档，回答“这个工程准备怎么做”。
2. `CIM标准知识体系/`
   面向通用读者的 `CIM / CGMES` 知识库，回答“标准是什么、为什么这样设计、工程上怎么理解”。

建议阅读顺序：

1. [当前状态.md](当前状态.md)
2. [design/README.md](design/README.md)
3. [CIM标准知识体系/README.md](CIM标准知识体系/README.md)

如果你的目标是进入 `lz-power` 项目开发，先看 `design/`。

如果你的目标是系统学习 `CIM / CGMES`，先看 `CIM标准知识体系/`。
